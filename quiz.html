HTML Quick Review & Tips

  • <h1> – <h6>: Headings, with <h1> the largest.
  • <p>: Defines paragraphs of text.
  • <a href="...">: Creates clickable links.
  • <img src="..." alt="...">: Adds images, use alt for accessibility.
  • <ul>, <ol>, <li>: Lists (unordered, ordered, list item).
  • <table>, <tr>, <td>, <th>: Tables and cells.
  • <form>: Used for user input forms.
  • <main>, <nav>, <footer>, <section>: Semantic tags for page structure.
  • <br>: Creates a line break.
  • <b>, <strong>: Makes text bold. <strong> is for importance.
  • <em>: Emphasizes text (usually italic).

Remember: HTML is all about structure. Use semantic tags for meaning, and always aim for accessible, well-organized code. Happy coding!
