CSS Variables & Advanced

1. CSS Variables (Custom Properties)

Define reusable values with --variable-name:

:root {
  --main-color: #2563eb;
  --secondary-color: #34d399;
  --padding: 10px;
}

.var-box {
  background: var(--secondary-color);
  color: #fff;
  padding: var(--padding);
}
Output:
This box uses CSS Variables

2. Pseudo-classes

Interactive states like :hover, :first-child, :nth-child():

.pseudo-hover {
  background: var(--highlight-example);
  padding: 10px;
  border: 2px solid var(--main-example);
  border-radius: var(--radius);
  transition: all 0.3s ease;
}
.pseudo-hover:hover {
  background: var(--main-example);
  color: var(--text);
  transform: scale(1.1);
}
Output:
Hover me!

4. Nested Variables & Hover Effects

Combine variables and hover for dynamic styling:

.nested-var {
  background: var(--soft-example);
  color: var(--main-example);
  border: 2px solid var(--main-example);
  padding: 10px;
  border-radius: var(--radius);
}
.nested-var:hover {
  background: var(--main-example);
  color: var(--highlight-example);
}
Output:
Hover me for effect

5. Practical Example: Card with Variable Colors

.card-example {
  --card-bg: #34d399;
  --card-text: #fff;
  background: var(--card-bg);
  color: var(--card-text);
  padding: 20px;
  border-radius: var(--radius);
  width: 200px;
  text-align: center;
  transition: all 0.3s ease;
}
.card-example:hover {
  --card-bg: #2563eb;
}
Output:
Hover Card

Variables make it easy to change multiple properties at once and create reusable styles.
