CSS Text & Fonts

Font Size

Font size defines the size of the text. You can use small, medium, large, or exact measurements like px, em, rem.

p.text-size-small { font-size: 0.9rem; }
p.text-size-medium { font-size: 1.2rem; }
p.text-size-large { font-size: 2rem; }

Font Family

CSS allows you to change the font family. Common choices include sans-serif, serif, and monospace.

p.text-family-sans { font-family: Arial, sans-serif; }
p.text-family-serif { font-family: 'Times New Roman', serif; }
p.text-family-mono { font-family: 'Courier New', monospace; }

Font Style & Weight

Use font-style to italicize or oblique text. Use font-weight to make text bold or bolder.

p.text-style-italic { font-style: italic; }
p.text-style-oblique { font-style: oblique; }
p.text-weight-bold { font-weight: bold; }
p.text-weight-bolder { font-weight: bolder; }

Text Color & Alignment

You can change the text color and align it left, center, or right using CSS.

p.text-color { color: #2563eb; }
p.text-align-left { text-align: left; }
p.text-align-center { text-align: center; }
p.text-align-right { text-align: right; }

Text Decoration & Transformation

Text can be underlined or have a line through it. You can also force text to be all uppercase or lowercase.

p.text-decoration-none { text-decoration: none; }
p.text-decoration-underline { text-decoration: underline; }
p.text-decoration-line-through { text-decoration: line-through; }
p.text-transform-uppercase { text-transform: uppercase; }
p.text-transform-lowercase { text-transform: lowercase; }
p.text-transform-capitalize { text-transform: capitalize; }

Line Height & Letter Spacing

You can control the vertical spacing between lines (line-height) and the horizontal spacing between letters (letter-spacing).

p.line-letter { line-height: 2; letter-spacing: 2px; }
