CSS Selectors

What are CSS Selectors?

CSS Selectors are patterns used to select HTML elements to apply styles. They are the core of styling in CSS.

Analogy: Think of selectors as ‚Äúaddresses‚Äù to elements: You need the right address to style the right element.

Types of Selectors with Examples

1. Universal Selector (*)

* {
  margin: 0;
  padding: 0;
}
Effect: Removes default margin and padding from all elements.

2. Type Selector (p, h1, div)

p.type {
  color: #2563eb;
  font-weight: bold;
}
Output:

This paragraph uses type selector (blue & bold)

3. Class Selector (.classname)

.class-example {
  color: green;
  font-style: italic;
}
Output:

This text uses a class selector (green & italic)

4. ID Selector (#idname)

#id-example {
  color: orange;
  font-size: 18px;
  font-weight: bold;
}
Output:

This text uses an ID selector (orange & bold)

5. Grouping Selectors (h1, h2, p)

h1, h2, p {
  font-family: Arial, sans-serif;
}
Effect: Applies the same font to multiple elements.

6. Descendant & Child Selectors

div p.descendant {
  color: purple;
}

div > p.child {
  color: teal;
}
Output:

This paragraph uses a descendant selector (purple)

This paragraph uses a child selector (teal)
