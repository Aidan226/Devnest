CSS Box Model

What is the Box Model?

Every HTML element is a rectangular box. The CSS Box Model describes the space it takes up, including content, padding, border, and margin.

/* Box Model Components */
.box {
  background-color: var(--edge);
  border: 2px solid var(--accent);
  padding: 1rem;
  margin: 1rem 0;
  width: 200px;
  text-align: center;
}
.box-padding { padding: 2rem; }
.box-border { border-width: 5px; }
.box-margin { margin: 2rem; }
