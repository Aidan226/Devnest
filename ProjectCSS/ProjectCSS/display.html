CSS Display

What is the Display Property?

The display property controls how an element is displayed on the page. It defines the layout behavior of the element, such as block, inline, flex, or grid.

/* Common Display Values */
.block        { display: block; }
.inline       { display: inline; }
.inline-block { display: inline-block; }
.flex         { display: flex; }
.grid         { display: grid; }

Block vs Inline

Block elements take full width and start on a new line. Inline elements flow with text.

<div class="display-box block">Block Element</div>
<span class="display-box inline">Inline Element</span>

Inline Element

Inline-Block

Inline-block elements sit inline like text, but allow width/height styling like block elements.

<span class="display-box inline-block">A</span>
<span class="display-box inline-block">B</span>
