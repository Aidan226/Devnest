CSS Transitions

1. Background Color Transition

Change background smoothly on hover:

<div class="transition-box bg-hover">Hover me</div>
Output:
Hover me

Explanation: The transition property smoothly changes the background-color over 0.5s.

2. Scale/Size Transition

Increase size smoothly on hover:

<div class="transition-box size-hover">Hover me</div>
Output:
Hover me

Explanation: The box smoothly grows larger using transform: scale().

3. Border Transition

Change border and background smoothly:

<div class="transition-box border-hover">Hover me</div>
Output:
Hover me

Explanation: Smooth transition for both border thickness and background color.

4. Multiple Property Transition

Combine multiple effects on hover:

<div class="transition-box multi-hover">Hover me</div>
Output:
Hover me

Explanation: Changes background, color, scale, and rotation simultaneously with smooth animation.

5. Custom Duration & Easing

Use different timing functions for unique effects:

.transition-custom {
  transition: all 1.5s ease-in-out;
}
<div class="transition-box transition-custom">Hover me slowly</div>
Output:
Hover me slowly

Explanation: You can control duration, delay, and easing function for smoother and custom animations.

6. Practical Example: Button Hover Effect

<button class="transition-box multi-hover" style="padding:10px 20px; font-size:1.1rem;">
  Click Me
</button>
Output:

Hovering the button changes color, scales up slightly, and rotates for a modern interactive effect.
