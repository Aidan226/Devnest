HTML Tables

Introduction

Tables organize data into rows and columns. They're essential for displaying things like schedules, comparisons, or results. Think of a table as a mini spreadsheet right on your webpage!

Basic Table Structure

Tables use <table> for the main container, <tr> for rows, and <td> for cells. 

<table>
  <tr>
    <td>Apple</td>
    <td>Banana</td>
  </tr>
  <tr>
    <td>Orange</td>
    <td>Grape</td>
  </tr>
</table>
Result:
AppleBanana
OrangeGrape

Table Headers: <th>

Use <th> for header cells (like column titles). They’re bold and let screen readers know it's a header. 

<table>
  <tr>
    <th>Name</th>
    <th>Score</th>
  </tr>
  <tr>
    <td>Alex</td>
    <td>95</td>
  </tr>
</table>
Result:
NameScore
Alex95

Adding a Caption

<caption> gives your table a title—very helpful for accessibility and clarity. 

<table>
  <caption>Top Students</caption>
  <tr><th>Name</th><th>Score</th></tr>
  <tr><td>Lila</td><td>99</td></tr>
</table>
Result:
Top Students
NameScore
Lila99

Spanning Rows & Columns

Use colspan to make a cell stretch across columns, and rowspan to stretch across rows. 

<table>
  <tr>
    <th colspan="2">Schedule</th>
  </tr>
  <tr>
    <td>Monday</td>
    <td>Math</td>
  </tr>
  <tr>
    <td rowspan="2">Tuesday</td>
    <td>Science</td>
  </tr>
  <tr>
    <td>Art</td>
  </tr>
</table>
Result:
Schedule
MondayMath
TuesdayScience
Art

Table Borders: HTML & CSS

Old HTML: border="1" (not recommended; use CSS!). 

<table border="1"> ... </table>
ExampleBordered

Modern CSS: Add borders and border-collapse: collapse; for a neat look. 

<table style="border:2px solid #5ee7ff; border-collapse:collapse;">
  <tr>
    <td style="border:1px solid #7c4dff;">Data</td>
    <td style="border:1px solid #7c4dff;">Data</td>
  </tr>
</table>
DataData

Styling Tables

  • :nth-child(even) for zebra stripes
  • :hover for row highlight
  • Colorful headers with th
  • Padding for readability
table {
  border-collapse: collapse;
  width: 100%;
}
th {
  background: #5fa8d3;
  color: #fff;
  padding: 0.8em;
}
td {
  padding: 0.8em;
}
tr:nth-child(even) td {
  background: #192135;
}
tr:hover td {
  background: #2b3266;
}

Tables with Links & Images

Table cells can contain <a> links, <img> images, or almost any HTML! 

<table>
  <tr>
    <th>Website</th>
    <th>Logo</th>
  </tr>
  <tr>
    <td><a href="https://github.com">GitHub</a></td>
    <td><img src="https://github.githubassets.com/images/modules/logos_page/GitHub-Mark.png" alt="GitHub" width="32"></td>
  </tr>
</table>
Result:
WebsiteLogo
GitHub

Accessibility Tips

  • Use <caption> for table titles
  • Use <th> for headers and add scope="col" or scope="row"
  • Don't use tables for layout—use for data only
<th scope="col">Score</th>
<th scope="row">Alex</th>

Common Mistakes

  • Forgetting <tr> (table rows)
  • Not using <th> for headers
  • Using tables for page layout (not for tabular data!)
  • Missing colspan or rowspan where needed

Real-world Examples

  • Schedules: Class times, conference agendas
  • Price lists: Product lists, menus
  • Product comparisons: Features across different items
  • Timetables: Bus, train, or flight times

Quick Recap: Table Cheat Sheet

HTML Elements:
  • <table>: The table container
  • <tr>: Table row
  • <td>: Table cell
  • <th>: Table header
  • <caption>: Table title
  • colspan, rowspan: Span columns/rows
CSS Styling Tips:
  • border-collapse: collapse; for clean borders
  • tr:nth-child(even) for zebra stripes
  • tr:hover for row highlight
  • th, td { padding: ... } for spacing
  • Use background for color
