Introduction to HTML

What is HTML?

HTML stands for HyperText Markup Language. It’s the basic building block for web pages.

Analogy: Imagine a website as a house:
  • HTML is the structure—the walls, rooms, and layout.
  • CSS is the paint and decorations—the colors, styles, and furniture.
  • JavaScript are the gadgets—the things that make your house interactive, like a remote-controlled light.

Your browser (like Chrome, Edge, or Firefox) reads the HTML file and shows the content as a web page.

Basic HTML Page Structure

<!DOCTYPE html>  <!-- Tells the browser this is an HTML5 document -->
<html>           <!-- The root of your HTML page -->
  <head>         <!-- Info about your page (title, links, etc) -->
    <title>My Page Title</title>
  </head>
  <body>         <!-- The visible content -->
    <!-- Content goes here -->
  </body>
</html>

Tags, Elements, and Attributes

  • Tag: The command inside <angle brackets>.
    Example: <p>
  • Element: The complete thing, including both opening and closing tags, and the content in between.
    Example: <p>Hello!</p>
  • Attribute: Extra info inside a tag.
    Example: <a href="https://example.com">Visit</a>. Here, href="..." is an attribute.

Hands-On Example

<!DOCTYPE html>
<html>
  <head>
    <title>My First HTML Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>This is my first HTML paragraph.</p>
  </body>
</html>
