Basic HTML Page Structure

<!DOCTYPE html>  <!-- Tells the browser this is an HTML5 document -->
<html>           <!-- The root of your HTML page -->
  <head>         <!-- Info about your page (title, links, etc) -->
    <title>My Page Title</title>
  </head>
  <body>         <!-- The visible content -->
    <!-- Content goes here -->
  </body>
</html>

Tags, Elements, and Attributes

  • Tag: The command inside <angle brackets>.
    Example: <p>
  • Element: The complete thing, including both opening and closing tags, and the content in between.
    Example: <p>Hello!</p>
  • Attribute: Extra info inside a tag.
    Example: <a href="https://example.com">Visit</a>. Here, href="..." is an attribute.
