HTML Structure

What is HTML Structure?

HTML structure is like the foundation of a building. It organizes content into a page.

Analogy: Think of HTML as a skeleton: <html> contains <head> and <body>.

Basic HTML Skeleton

<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    Page content goes here.
  </body>
</html>
