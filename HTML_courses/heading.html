HTML Headings & Paragraphs

Introduction: What are Headings & Paragraphs?
Headings are titles that organize your web page into sections, like chapters in a book.
Paragraphs are blocks of text that share information, stories, or ideas.

Both make your content easy to read and understand!

In HTML, headings use tags <h1> through <h6>, and paragraphs use the <p> tag.

Headings: <h1> to <h6>

Headings tell readers what a section is about. There are six levels:

<h1>This is Heading 1</h1>
<h2>This is Heading 2</h2>
<h3>This is Heading 3</h3>
<h4>This is Heading 4</h4>
<h5>This is Heading 5</h5>
<h6>This is Heading 6</h6>

This is Heading 1

This is Heading 2

This is Heading 3

This is Heading 4

This is Heading 5
This is Heading 6

Tip: Use <h1> for your main title, then smaller headings for subsections.

Paragraphs: <p>

Paragraphs hold sentences and ideas. Each <p> creates a new block of text with space around it.

<p>HTML makes websites easy to read.</p>
<p>This is another paragraph.</p>

HTML makes websites easy to read.

This is another paragraph.

Use paragraphs to separate thoughts or topics for better readability.

Line Breaks: <br>

The <br> tag adds a single line break — like pressing "Enter" once. Use it inside paragraphs when you need a new line, but not a whole new paragraph.

<p>
123 Main St.<br>
Springfield, IL<br>
USA
</p>

123 Main St.
Springfield, IL
USA

Best for: Addresses, poems, or forms where you need lines stacked together.

Combining Headings & Paragraphs: A Short Article
<h1>All About Space</h1>
<h2>The Solar System</h2>
<p>Our solar system has eight planets.</p>
<h3>Fun Fact</h3>
<p>Jupiter is the largest planet!</p>

All About Space

The Solar System

Our solar system has eight planets.

Fun Fact

Jupiter is the largest planet!

Styling Headings & Paragraphs (inline CSS)
<h2 style="color:#2563eb;">Blue Heading</h2>
<p style="color:#38bdf8;
          text-align:center;
          font-size:1.1rem;">
This is a centered, blue paragraph!
</p>

Blue Heading

This is a centered, blue paragraph! 

<h3 style="font-size:2rem;
       font-family:monospace;">
Monospace Heading
</h3>

Monospace Heading

Inline CSS lets you style each element directly. For bigger projects, use separate CSS files!

Best Practices
  • Use only one <h1> per page – this is your main title.
  • Use headings in order: start with <h2> for major sections, then <h3> for subsections.
  • Don’t skip heading levels (<h2> ➔ <h4> is bad).
  • Keep paragraphs short for readability.
  • Don’t use headings just to make things big or bold — use them for structure!
  • Don’t rely on <br> for layout; use paragraphs for new thoughts.
Accessibility & SEO Tips
  • Headings help screen readers and users navigate your page.
  • Google uses headings to understand your content — good for SEO!
  • Write meaningful headings, not just “Welcome” or “Stuff”.
  • Use paragraphs for real text, not headings for everything.
Common Mistakes: Bad vs Good
Bad Example: 
<h1>Welcome!</h1>
<h1>Products</h1>  <!-- Two h1s (bad) -->
<p><br>Paragraph with too many <br>s</p>
Good Example: 
<h1>Welcome!</h1>
<h2>Products</h2>  <!-- Use h2 for section -->
<p>A proper paragraph.</p>

Tip: Always use headings in order, and paragraphs for text blocks. Only one <h1> per page!

Quick Recap (Cheat Sheet)
  • <h1> – <h6>: Headings, organize content. Only one <h1>!
  • <p>: Paragraph, for blocks of text.
  • <br>: Line break, for stacking lines inside paragraphs.
  • Use headings in order for structure and accessibility.
  • Headings & paragraphs help SEO and screen readers.
  • Don’t use headings just for bigger text — use them for meaning!
