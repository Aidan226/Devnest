HTML Links

Introduction

🔗 Analogy: Links are like bridges that let you jump between places on the web.

Links (or "hyperlinks") are used to connect different web pages, websites, or resources. They let users move easily from one place to another!

Basic Link Structure

The <a> tag creates a link. You must use the href attribute to tell where the link goes.

<a href="https://example.com">Visit Example</a>

<a> is the anchor (link) tag.
href is where the link points to.
The text between the tags is what people click.

Absolute vs. Relative Links

Absolute links go to a full address (like another website).
Relative links go to another page or file in your own website. 

<!-- Absolute link -->
<a href="https://wikipedia.org">Wikipedia</a>

<!-- Relative link -->
<a href="about.html">About Us</a>

Open in a New Tab

Add target="_blank" to open a link in a new tab.

<a href="https://google.com" target="_blank">Google</a>

Tip: For security, add rel="noopener" for external links.

<a href="https://google.com" target="_blank" rel="noopener">Google</a>

Email & Phone Links

You can make links that open the user's email app or call a phone number!

<a href="mailto:hello@example.com">Email us</a>
<a href="tel:+1234567890">Call us</a>

Anchor Links (Jump on Same Page)

Use anchor links to jump to another part of the same page. The href uses #id and the target section gets an id. 

<a href="#contact">Contact Section</a>
...
<h2 id="contact">Contact Us</h2>

Linking Images

Make an image clickable by putting it inside an <a> tag.

<a href="https://html.com">
  <img src="logo.png" alt="HTML Logo">
</a>

Styling Links with CSS

You can style links for normal, hover, visited, and active states!

a {
  color: #5ee7ff;
  text-decoration: underline;
}
a:hover {
  color: #7c4dff;
  background: #e7f0fe;
}
a:visited {
  color: #8e44ad;
}
a:active {
  color: #e17055;
  background: #fff3e0;
}

Best Practices

  • Use clear, descriptive link text (not just “click here”).
  • Make links easy to see and obvious when hovered.
  • For accessibility, describe where a link goes.
  • Use target="_blank" sparingly and warn users if needed.
  • Test your links—they should all work!

Common Mistakes

<a>Google</a>   ← Missing href!
<a href="https://google.com">Google</a>
<a href="page.html">Click here</a>   ← Not descriptive
<a href="page.html">Read our About Page</a>

Real-World Examples

  • Navigation menu: <a href="about.html">About</a>
  • Back to top: <a href="#top">Back to top</a>
  • External resource: <a href="https://developer.mozilla.org/" target="_blank">MDN Docs</a>
  • Download link: <a href="file.pdf" download>Download PDF</a>

Quick Recap: Cheat Sheet

  • <a href="url">...</a> - Basic link
  • href – Link destination (URL, file, #id, mailto:, tel:)
  • target="_blank" – Open in new tab
  • mailto:someone@domain.com – Open email app
  • tel:+number – Call via phone
  • #id – Jump to section with that id
  • download – Download file
