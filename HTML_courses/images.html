HTML Images

Introduction

Images add beauty, information, and life to your web pages. They help illustrate ideas, show products, and make your site more engaging!

Basic Syntax: <img>

The <img> tag inserts an image. The two most important attributes are:

  • src: the path or URL to the image file
  • alt: alternative text (shows if the image canâ€™t load, and is used by screen readers)
<img src="https://placekitten.com/200/120" alt="A cute cat">

Image Attributes: width, height, title

Add width or height to control size, and title for a tooltip.

<img src="https://picsum.photos/120/80" alt="A random image" width="120" height="80">
<img src="https://picsum.photos/120/80" alt="A random image" title="Random!">

Image Sources: External

External image: an image from another website (URL).

<img src="https://picsum.photos/200/120" alt="Random image">

Note: Only use images you are allowed to. Donâ€™t steal â€” use your own or free/open images!

Image Formats

  • JPG/JPEG: Good for photos. Small file, loses some quality.
  • PNG: Good for logos, icons, graphics with transparency. Bigger file size.
  • GIF: Simple animations, few colors.
  • SVG: Vector graphics, icons, scales to any size.
  • WebP: Modern, small size, good for web.

Resizing Images: HTML vs CSS

<!-- HTML width/height -->
<img src="https://picsum.photos/250/170" width="120" height="80">

<!-- CSS width/height -->
<img src="https://picsum.photos/250/170" style="width:120px; height:80px;">

Tip: To keep images looking good, set only width or use height: auto; in CSS.

Responsive Images

img {
  max-width: 100%;
  height: auto;
}

This CSS makes images shrink on small screens so they never overflow!

Styling Images with CSS

img {
  border: 3px solid #5ee7ff;
  border-radius: 12px;
  box-shadow: 0 2px 12px #b2bdd066;
}

Hover effect: Try moving your mouse over the images!
