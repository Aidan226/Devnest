If / Else Statements & Loops

If / Else

Used to run code based on a condition.

let age = 20;

if(age >= 18) {
  console.log("You are an adult.");
} else {
  console.log("You are a minor.");
}
Sample Output:
You are an adult.

If / Else If / Else

let score = 85;

if(score >= 90) {
  console.log("Grade: A");
} else if(score >= 75) {
  console.log("Grade: B");
} else {
  console.log("Grade: C");
}
Sample Output:
Grade: B

Loops

Loops are used to repeat code multiple times:

  • for loop â†’ repeats a block of code a known number of times
  • while loop â†’ repeats while a condition is true
  • do...while loop â†’ executes at least once, then repeats while condition is true

Example: For Loop

for(let i = 1; i <= 5; i++) {
  console.log("Number: " + i);
}
Sample Output:
Number: 1
Number: 2
Number: 3
Number: 4
Number: 5

Example: While Loop

let i = 1;
while(i <= 3) {
  console.log("Count: " + i);
  i++;
}
Sample Output:
Count: 1
Count: 2
Count: 3

Example: Do...While Loop

let j = 1;
do {
  console.log("Step: " + j);
  j++;
} while(j <= 2);
Sample Output:
Step: 1
Step: 2
