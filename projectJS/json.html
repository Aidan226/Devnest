JavaScript JSON

1. What is JSON?

JSON (JavaScript Object Notation) is a lightweight format for storing and exchanging data. It is easy to read and write for humans and easy to parse and generate for machines.

JSON is often used when sending data between a client and server.

2. JSON Syntax

  • Data is in key/value pairs.
  • Keys must be strings wrapped in double quotes.
  • Values can be strings, numbers, objects, arrays, booleans, or null.
  • Objects are wrapped in {}, arrays in [].

3. JSON Example

let personJSON = '{"name": "Alice", "age": 25, "city": "New York"}';
console.log(personJSON);
Sample Output:
{"name": "Alice", "age": 25, "city": "New York"}

4. Converting JSON to JavaScript Object

let personObj = JSON.parse(personJSON);
console.log(personObj.name);
console.log(personObj.age);
Sample Output:
Alice
25

5. Converting JavaScript Object to JSON

let student = {
  name: "Bob",
  grade: "A",
  subjects: ["Math", "English"]
};
let studentJSON = JSON.stringify(student);
console.log(studentJSON);
Sample Output:
{"name":"Bob","grade":"A","subjects":["Math","English"]}

6. Nested JSON Example

let dataJSON = '{"team": "Red Dragons","players":[{"name":"Alice","position":"Forward"},{"name":"Bob","position":"Goalkeeper"}]}';
let dataObj = JSON.parse(dataJSON);
console.log(dataObj.team);
console.log(dataObj.players[0].name);
Sample Output:
Red Dragons
Alice

7. Tips

  • Always use double quotes for keys in JSON.
  • Use JSON.parse() to convert JSON into objects you can manipulate in JavaScript.
  • Use JSON.stringify() to prepare JavaScript objects for sending to a server.
  • Nested objects and arrays are supported in JSON.
