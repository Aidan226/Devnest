Introduction to JavaScript

What is JavaScript?

JavaScript (JS) is the programming language of the web. It lets you make pages interactive â€” responding to clicks, showing/hiding content, fetching data, and more.

Analogy: If a website were a house:
  • HTML = the structure (walls, rooms)
  • CSS = the style (paint, decorations)
  • JavaScript = the behavior (lights, doors, gadgets)

Simple Example

Hereâ€™s a minimal HTML page that runs a small JavaScript snippet:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8" />
  <title>My First JS</title>
  <script>
    // This runs as soon as the page loads
    console.log("Hello from JavaScript!");
    alert("Welcome to my page!");
  </script>
</head>
<body>
  <h1>Hello, World!</h1>
</body>
</html>
Sample Output:
  • A popup alert box appears saying: Welcome to my page!
  • The browser console logs: Hello from JavaScript!
  • The page displays: Hello, World!
