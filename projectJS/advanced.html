Advanced JavaScript Topics

1. The this Keyword

console.log(this);

const obj = {
  name: "Alice",
  sayThis: function() {
    console.log(this);
  }
};
obj.sayThis();
Explanation: In global scope, this refers to window. Inside an object method, it refers to the object itself.

2. Classes

class Animal {
  constructor(name) {
    this.name = name;
  }
  speak() {
    console.log(this.name + " makes a sound.");
  }
}

const dog = new Animal("Dog");
dog.speak();
Explanation: Classes allow creating objects with constructors and methods. The this inside class refers to the instance.

3. Promises

const url = "https://jsonplaceholder.typicode.com/todos/1";

fetch(url)
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(err => console.error("Error:", err));
Explanation: Promises handle asynchronous operations. then() executes on success, catch() on failure.

4. Async / Await

async function getData() {
  try {
    let res = await fetch(url);
    let data = await res.json();
    console.log(data);
  } catch(err) {
    console.error(err);
  }
}

getData();
Explanation: async/await makes asynchronous code look synchronous. Use try/catch to handle errors.

5. Local Storage

localStorage.setItem("user","Alice");
console.log(localStorage.getItem("user"));
localStorage.removeItem("user");
Explanation: Store key-value data in browser. Data persists across sessions until removed.

6. Geolocation API

navigator.geolocation.getCurrentPosition(pos => {
  console.log("Latitude:", pos.coords.latitude);
  console.log("Longitude:", pos.coords.longitude);
});
Explanation: Get user's current position. Useful for location-based apps.

7. Canvas API

const canvas = document.getElementById("myCanvas");
const ctx = canvas.getContext("2d");

ctx.fillStyle = "#2563eb";
ctx.fillRect(10,10,100,50);
ctx.fillStyle = "#fff";
ctx.font = "16px Arial";
ctx.fillText("Canvas Box", 15, 40);
Explanation: Draw rectangles, shapes, text, and graphics on a webpage using Canvas.
