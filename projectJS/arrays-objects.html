JavaScript Arrays & Objects

1. Arrays

Arrays are ordered lists of values. They can store multiple items in a single variable. Arrays are zero-indexed (first element has index 0).

Example 1: Creating and Accessing Arrays

let fruits = ["Apple", "Banana", "Cherry"];
console.log(fruits[0]); // Apple
console.log(fruits[2]); // Cherry
console.log(fruits.length); // 3
Sample Output:
Apple
Cherry
3

Example 2: Adding and Removing Elements

fruits.push("Durian");   // Add to end
fruits.unshift("Mango"); // Add to start
fruits.pop();            // Remove last
fruits.shift();          // Remove first
console.log(fruits);     // Current array
Sample Output:
[ 'Apple', 'Banana', 'Cherry' ]

Example 3: Looping Through an Array

for(let i = 0; i < fruits.length; i++) {
  console.log(fruits[i]);
}

for(let fruit of fruits) {
  console.log(fruit);
}
Sample Output:
Apple
Banana
Cherry
Apple
Banana
Cherry

Example 4: Useful Array Methods

let numbers = [1, 2, 3, 4, 5];

// map: multiply each element by 2
let doubled = numbers.map(num => num * 2);

// filter: keep only even numbers
let evens = numbers.filter(num => num % 2 === 0);

// slice: get part of array
let part = numbers.slice(1, 4);

// splice: remove 1 element at index 2 and add 10
numbers.splice(2, 1, 10);

console.log(doubled);
console.log(evens);
console.log(part);
console.log(numbers);
Sample Output:
[2, 4, 6, 8, 10]
[2, 4]
[2, 3, 4]
[1, 2, 10, 4, 5]

2. Objects

Objects store data as key-value pairs. Keys (properties) are strings, and values can be anything (numbers, strings, arrays, other objects, functions).

Example 1: Creating and Accessing Objects

let person = {
  name: "Alice",
  age: 25,
  city: "New York"
};

console.log(person.name);    // Alice
console.log(person["age"]);
